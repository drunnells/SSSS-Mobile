SSSS Mobile v0 by Dustin Runnells

Split


The parameters on this tab will let you create a set of new shares.

Secret Message: The message to be split. This must be a short message of less than 128 characters long. If you need to use this tool on anything larger, consider encrypting with a strong PGP/GPG symmetric cipher and SSSS splitting only the password.

Threshold: This is the exact number of shares required to reassemble your secret message. Any combination of this number of shares will allow you to retrieve your message. Remember this number as you will be unable to reassemble your secret message without knowing the exact threshold needed for the set.

Shares: This is the total number of shares generated. The total number of shares must be equal to or larger than the threshold.

Token: The Token is optional short text string that will prefix all SSSS shares in this set. The Token may be used to indicate the significance of the message contents and/or the threshold required for retrieval.

Once your shares have been generated they will appear on the bottom half of the screen. Shares can be copied to the clipboard or shared. Clicking the "share" icon will give you a QR Code representation of the current share, an option to email the share and QR Code and an option to write the share to an NFC tag (not available on all devices).

Combine


The Combine tab has only one text area to enter all SSSS shares needed to reassemble a message. Shares can be pasted into the text area, retrieved from QR Code or scanned from NFC tags (not available on all devices). Note that you must enter exactly the correct number of shares set by the threshold to retrieve the split message. Too many or too few shares will result an an error OR incorrect secret message.
